# 🧠 CogniScan AI  
### AI-Powered Dementia Screening & Cognitive Assessment System

CogniScan AI is a healthcare-focused web application designed for early dementia screening using cognitive evaluation, functional assessment, and dementia risk analysis.  

The platform provides an interactive assessment experience, real-time scoring, performance analytics, and downloadable medical-style reports.

---

# 🚀 Features

## 🧠 Cognitive Assessment (MMSE-Inspired)
- Orientation questions
- Memory recall tasks
- Attention & calculation tests
- Language evaluation
- Typing-based interaction
- Automatic scoring

---

## 🧾 Functional Assessment
Evaluates daily functional abilities through:
- Memory-related tasks
- Navigation ability
- Financial management
- Medication handling
- Daily routine assessment

Includes:
- MCQ-based interaction
- Severity classification
- Impairment analysis

---

## ⚠️ Dementia Risk Analysis
Analyzes common dementia risk factors such as:
- Age
- Family history
- Diabetes
- Hypertension
- Smoking
- Sleep quality
- Physical activity

Provides:
- Low / Moderate / High risk interpretation

---

## 📊 Analytics Dashboard
- Interactive charts
- Progress tracking
- Performance overview
- Test filtering
- Color-coded analysis
- Score normalization (/100)

---

## 📄 PDF Report Generation
Downloadable professional report including:
- User details
- Test history
- Graphical analysis
- Functional status
- Risk interpretation
- Suggestions & recommendations

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/SAJNA12345/CogniScan-AI.git
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# ▶️ Run Application

## Start Backend

```bash
npm start
```

Runs on:
```bash
http://localhost:5000
```

---

## Start Frontend

```bash
npm run dev
```

Runs on:
```bash
http://localhost:5173
```
